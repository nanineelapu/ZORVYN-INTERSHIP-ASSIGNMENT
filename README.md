# Finance Dashboard UI ZORVYN INTERSHIP ASSESMENT

## Overview
This project is a simple finance dashboard built to visualize financial activity using mock data. It allows users to view summary metrics, analyze spending patterns, and explore transactions.

## Features
- Summary cards (Balance, Income, Expenses)
- Charts (Balance trend and category breakdown)
- Transactions table with search, filter, and sorting
- Role-based UI (Admin / Viewer)
- Insights (Savings, top category, burn rate)

## Tech Stack
- React
- Tailwind CSS
- Recharts

## State Management
Used React Context API to manage:
- Transactions data
- Role switching
- Search and filters

## How to Run
npm install  
npm run dev

## Approach
Focused on building a clean and intuitive UI while keeping the implementation simple and modular. Used mock data and basic calculations to simulate real-world financial insights.
